# 기능 요구사항

### 필수

- [x] 다음의 값을 입력받을 수 있다.

  - [x] 카드 번호
    - 뒤의 8자리 값은 가려져야 한다.
  - [x] 만료일
  - [x] 카드 소유자 이름(선택)
  - [x] 보안 코드(CVC/CVV)
    - 이 값은 가려져야 한다.
  - [x] 카드 비밀번호(앞 2자리만)
    - 이 값은 가려져야 한다.

- [x] 입력값에 맞게 카드 정보를 보여줄 수 있다.

  - 카드 번호
  - 만료일
    - 초기값은 `MM/YY`이다.
  - 카드 소유자 이름
    - 초기값은 `NAME`이다.

- [x] `?`를 클릭하면 보안 코드에 대한 설명을 보여줄 수 있다.
  - 모달 형식
